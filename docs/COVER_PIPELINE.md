# Jaquettes

La cible autorisée est `libretro-thumbnails/Sega_-_Mega_Drive_-_Genesis`, dossier `Named_Boxarts`. Aucun hotlinking à l'exécution. Seuls les titres/alias exacts et non ambigus doivent être acceptés; priorité Europe, World, USA, Japan. `npm run covers:sync` initialise les rapports. Les images doivent être validées, puis produites en WebP 420×588 qualité 76 et JPEG progressif qualité 78 avec Sharp. En l'absence de match sûr, le placeholder RetroCoop est affiché.
