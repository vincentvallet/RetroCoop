# Sources des jaquettes

Les jaquettes locales de Retro Coop sont récupérées depuis le dépôt public `libretro-thumbnails/Sega_-_Mega_Drive_-_Genesis`, exclusivement dans son dossier `Named_Boxarts`, puis converties en WebP. Le dépôt source complet n’est pas distribué avec l’application.

La disponibilité publique du dépôt ne signifie pas que les illustrations de boîtes sont libres de droits. Les marques, titres et visuels appartiennent à leurs ayants droit respectifs. RetroCoop est un projet expérimental gratuit indépendant et sans affiliation.

Chaque import conserve le nom du fichier source, la région, le score de rapprochement et la date dans `public/covers/megadrive/covers-manifest.json`. La date exacte du dernier import figure dans ce manifeste.

Pour demander une correction ou un retrait : contact@ofid.fr.
